# Utilizar os exemplos


## 1)Instalar todas as dependências

```
npm install
```

## 2)Configurar o serviço
criar arquivo .env com o seguinte conteúdo : 

```
REACT_APP_REST_HOST = http://127.0.0.1:3002
```

## 3)Executar serviço

```
npm start
```

## 4)Abrir página

Abrir browser no endereço http://localhost:3000
